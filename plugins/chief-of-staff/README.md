# Chief of Staff

Reusable Codex guidance for creating a quiet chief-of-staff system with:

- a visible pinned HQ thread;
- read-only recurring automation lanes;
- local-time run-window guards;
- no-noise reporting rules;
- private profile values supplied by a separate local/private setup plugin.

This plugin intentionally does not ship live automation state, thread IDs, credentials, or user-specific paths.
